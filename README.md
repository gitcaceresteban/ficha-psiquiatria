# Ficha Clínica · Psiquiatría

Plantilla interactiva para evaluación psiquiátrica diseñada para uso clínico estudiantil. Genera automáticamente texto narrativo para ficha clínica a partir de selecciones rápidas tipo chip. A diferencia de calculadoras aisladas como MDCalc o sistemas empresariales como PsyQuation, esta plantilla está diseñada específicamente para estudiantes de medicina en rotación de psiquiatría en Chile, con terminología MINSAL y generación automática de narrativa clínica lista para copiar en la ficha del paciente.

## ✨ Características

- 📋 **Anamnesis completa**: identificación, anamnesis próxima y remota
- 🧠 **Examen mental** estructurado en 10 ítems + apariencia
- 💬 **Tooltips clínicos** sobre conceptos clave (estereotipias, taquifemia, bradidipsia, etc.)
- ⚙ **Generación automática** de texto narrativo cohesionado para copiar/pegar
- 💾 **Autoguardado** en el navegador (los datos no se pierden al recargar)
- 📄 **Descarga PDF** con un clic
- 🖨 **Optimizada para impresión**
- 📱 **Responsive** — funciona en móvil, tablet y escritorio
- 🔒 **100% local** — ningún dato se envía a servidores externos

## 🚀 Cómo usar

1. Completa los campos de identificación y anamnesis
2. Marca los chips correspondientes a cada ítem del examen mental
3. Pasa el cursor sobre términos con tooltip para ver su definición
4. Presiona **⚙ Generar texto** para obtener los párrafos narrativos
5. Usa **📋 Copiar texto** para pegar en la ficha clínica del paciente
6. Usa **📄 Descargar PDF** para guardar el formulario completo

## ⚖️ Uso

Material educativo de apoyo para estudiantes de medicina en rotación de psiquiatría. No reemplaza el juicio clínico ni la supervisión docente.
